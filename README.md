# SwiftSPICE

**SwiftSPICE** is a Swift package that provides an interface for working with the SPICE toolkit, a collection of software tools and data for space science applications, primarily developed by NASA's Jet Propulsion Laboratory (JPL). With SwiftSPICE, you can integrate SPICE functionality into your Swift-based projects to analyze and manipulate space science data.
